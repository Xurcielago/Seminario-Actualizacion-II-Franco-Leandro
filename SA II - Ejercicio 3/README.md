# # Proyecto: Registro de Alumnos

Aplicación para cargar alumnos con nombre, edad y nota. Los muestra ordenados por nota (mayor a menor) y nombre como desempate. Tiene dos versiones.

---

## Parte 1 — Solo navegador

## Abrir el archivo `parte1/index.html` directamente en el navegador. No requiere instalación. Los datos se guardan en el navegador y persisten aunque se cierre

## Parte 2 — Backend Node.js

Requiere [Node.js](https://nodejs.org/) instalado.

**Iniciar el servidor:**

```bash
node server.js
```

El servidor queda corriendo en el puerto `3000`. Para acceder, abrir en el navegador:

```
http://localhost:3000
```

Desde otra computadora en la misma red, reemplazar `localhost` por la IP de la máquina que corre el servidor.
